HTML

introduccion


¡Hola! puedo decirte que HTML es un lenguaje que usamos para crear páginas web. Es como un "esqueleto" de la página que nos ayuda a organizar todo el contenido que queremos mostrar.

Imagínate que estás construyendo una casa: HTML sería como el armazón de la casa, que sirve de soporte para todo lo que quieres poner dentro, como las habitaciones, la cocina, los baños, etc.

Los elementos de HTML son como bloques de construcción que usamos para crear nuestra página web. Cada uno tiene una etiqueta de apertura y una etiqueta de cierre, como si fueran un par de corchetes < >. Por ejemplo, para escribir un párrafo, usamos la etiqueta <p> al principio y </p> al final.

También tenemos elementos que nos ayudan a estructurar el contenido de la página, como el encabezado (head) y el cuerpo (body). El encabezado contiene información importante sobre la página, como el título y enlaces a otros archivos, mientras que el cuerpo es donde ponemos todo el contenido que queremos mostrar, como texto, imágenes, videos, etc.

En resumen, HTML es un lenguaje muy importante para crear páginas web, que nos ayuda a organizar todo el contenido que queremos mostrar. ¡Es como construir una casa en línea!

Ejemplos
<html> Define el inicio y el final del documento HTML <html>...</html>
<head> Contiene información sobre el documento HTML <head>...</head>
<title> Define el título del documento HTML <title>Mi página web</title>
<body> Contiene el contenido visible del documento HTML <body>...</body>
<h1>...<h6> Define encabezados de diferentes niveles <h1>Encabezado de nivel 1</h1>
<h2>Encabezado de nivel 2</h2>
<p> Define un párrafo <p>Este es un párrafo de ejemplo.</p>
<a> Define un enlace <a href="https://www.google.com">Ir a Google</a>
<img> Define una imagen <img src="mi_imagen.png" alt="Descripción de la imagen">
<ul> Define una lista sin orden <ul>
<li>Elemento 1</li>
<li>Elemento 2</li>
</ul>
<ol> Define una lista ordenada <ol>
<li>Primer elemento</li>
<li>Segundo elemento</li>
</ol>
<div> Define una sección de la página que se puede estilizar con CSS o manipular con JavaScript. <div id="contenedor">Contenido aquí</div>
<span> Define una sección de texto que se puede estilizar con CSS o manipular con JavaScript. <p>Este es un texto de ejemplo con un <span style="color: red">texto rojo</span> dentro.</p>
<br> Crea una línea nueva dentro de un párrafo. <p>Este es un párrafo.<br>Esta es una nueva línea dentro del mismo párrafo.</p>
<hr> Crea una línea horizontal que separa secciones de la página. <h2>Sección 1</h2>
<p>Contenido de la sección 1.</p>
<hr>
<h2>Sección 2</h2>
<p>Contenido de la sección 2.</p>
<strong> Hace que el texto sea más enfático o importante. <p>Este es un texto <strong>importante</strong>.</p>
<em> Hace que el texto sea enfático o destacado. <p>Este es un texto <em>destacado</em>.</p>
<blockquote> Crea una cita de texto que se muestra en un bloque separado. <blockquote>Este es un texto citado.</blockquote>
<cite> Indica la fuente de una cita. <blockquote>Este es un texto citado. <cite>Autor de la cita</cite></blockquote>
<form> Crea un formulario para que los usuarios ingresen datos o envíen información al servidor. <form action="procesar.php" method="POST">
<label for="nombre">Nombre:</label>
<input type="text" id="nombre" name="nombre">
<input type="submit" value="Enviar">
</form>
<input> Crea un campo de entrada de formulario para que los usuarios ingresen datos. <label for="correo">Correo electrónico:</label>
<input type="email" id="correo" name="correo">
<textarea> Crea un campo de entrada de formulario para que los usuarios ingresen texto largo. <label for="mensaje">Mensaje:</label>
<textarea id="mensaje" name="mensaje"></textarea>
<button> Crea un botón que se puede hacer clic para realizar una acción. <button type="button" onclick="alert('Hola mundo!')">Haz clic aquí</button>
<table> Crea una tabla con filas y columnas. <table>
<tr>
<td>Celda 1</td>
<td>Celda 2</td>
</tr>
</table>
<thead> Crea la sección de encabezado de una tabla. <table> <thead> <tr> <th>Encabezado 1</th> <th>Encabezado 2</th> </tr> </thead> <tbody> <tr> <td>Celda 1</td> <td>Celda 2</td> </tr> </tbody> </table>
<tbody> Crea la sección de cuerpo de una tabla. <table> <thead> <tr> <th>Encabezado 1</th> <th>Encabezado 2</th> </tr> </thead> <tbody> <tr> <td>Celda 1</td> <td>Celda 2</td> </tr> </tbody> </table>
<tfoot> Crea la sección de pie de una tabla. <table> <thead> <tr> <th>Encabezado 1</th> <th>Encabezado 2</th> </tr> </thead> <tbody> <tr> <td>Celda 1</td> <td>Celda 2</td> </tr> </tbody> <tfoot> <tr> <td>Total</td> <td>$100</td> </tr> </tfoot> </table>
<th> Crea una celda de encabezado en una tabla. <table> <thead> <tr> <th>Encabezado 1</th> <th>Encabezado 2</th> </tr> </thead> <tbody> <tr> <td>Celda 1</td> <td>Celda 2</td> </tr> </tbody> </table>
<col> Define las propiedades de una o varias columnas de una tabla. <table> <colgroup> <col span="2" style="background-color:red;"> </colgroup> <tr> <td>Celda 1</td> <td>Celda 2</td> </tr> </table>
<colgroup> Agrupa las etiquetas <col> para definir las propiedades de varias columnas de una tabla. <table> <colgroup> <col span="2" style="background-color:red;"> </colgroup> <tr> <td>Celda 1</td> <td>Celda 2</td> </tr> </table>
<caption> Define el título o leyenda de una tabla. <table> <caption>Mi tabla</caption> <tr> <td>Celda 1</td> <td>Celda 2</td> </tr> </table>
article Define un artículo independiente dentro de una página web <article> <h2h2>Título del artículo </h2> <ph2>Contenido del artículo </p> </article>
section Define una sección lógica dentro de una página web <section> <h2>Título de la sección </h2> <p>Contenido de la sección </p> </section>
aside Define un contenido adicional relacionado con el contenido principal de una página web <article> <h2>Título del artículo </h2> <p>Contenido del artículo </p> </article> <aside> <h3>Contenido adicional </h3> <p>Contenido relacionado con el artículo </p> </aside>
footer Define la sección de pie de página de una página web <footer> <p>Derechos de autor © 2023 - Mi sitio web </p> </footer>
main Define el contenido principal de una página web <main> <h1>Título de la página </h1> <p>Contenido principal de la página </p> </main>
nav Define un grupo de enlaces de navegación <nav> <ul> <li> <a href="#">Inicio </a> </li> <li> <a href="#">Acerca de </a> </li> <li> <a href="#">Contacto </a> </li> </ul> </nav>
